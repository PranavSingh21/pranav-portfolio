# Savvy — Product Brief

## Overview

Savvy is an AI-first conversational personal finance assistant designed to make money tracking frictionless.

Instead of forcing users into rigid forms, filters, and budgeting dashboards, Savvy lets users log expenses and ask financial questions in plain English — the same way they would text a friend.

Examples:
- “Swiggy 280”
- “Salary 72000”
- “How much did I spend on groceries?”
- “Can I spend 2k this weekend?”

Savvy interprets intent, classifies the transaction, updates financial memory, and responds with lightweight contextual guidance.

The goal is not full-scale bookkeeping.  
The goal is fast, low-friction financial awareness.

---

## Problem

Most personal finance tools fail because they introduce too much effort too early.

Users are expected to:
- open an app
- select transaction type
- choose category
- enter amount
- add notes
- save manually
- later interpret charts

This creates friction for the simplest task: logging money.

Most users do not need accounting software.  
They need a fast way to answer:

- “Where did my money go?”
- “Can I afford this?”
- “Am I overspending?”

Savvy reduces the cost of financial awareness by turning expense tracking into conversation.

---

## Target User

Savvy is designed for young professionals and digitally active users who:
- earn monthly income
- spend across multiple informal categories
- do not maintain structured budgets
- want lightweight visibility into spending
- prefer conversational interfaces over dashboards

Primary user archetype:
- salaried
- mobile-first
- spends across food, cabs, subscriptions, groceries, and impulse categories
- wants control without complexity

---

## Core Value Proposition

Savvy helps users track money with less effort and more context.

It replaces:
- manual logging
- rigid categories
- spreadsheet thinking

with:
- natural language input
- smart classification
- contextual spend awareness

Savvy is not trying to be accounting software.  
It is trying to be a lightweight money copilot.

---

## MVP Scope

Savvy MVP includes:

### 1. Conversational Expense Logging
Users can log spending in natural language.

Examples:
- “Swiggy 280”
- “Ola 450”
- “Groceries 1200”
- “Chicken masala powder 300”

Savvy extracts:
- amount
- merchant
- category
- intent

and logs the transaction.

---

### 2. Conversational Financial Queries
Users can ask natural questions like:
- “How much did I spend this month?”
- “Groceries total”
- “Transport total”
- “Spending breakdown”
- “Safe to spend”

Savvy returns direct answers without requiring navigation.

---

### 3. AI-Assisted Intent Parsing
Savvy uses AI to classify user input into:
- expense_add
- income_add
- savings_add
- expense_query
- unknown

This allows flexible user input while preserving structured state.

---

### 4. Lightweight Financial Memory
Savvy stores:
- transaction history
- salary
- savings
- balance
- inferred spend categories

This powers contextual responses and spend summaries.

---

### 5. Real-Time Spend Summary
Savvy continuously updates:
- current balance
- total spend
- category totals
- top category
- safe-to-spend estimate

This gives users passive financial visibility while chatting.

---

## Non-Goals (MVP)

Savvy intentionally does not include:

- bank integrations
- account syncing
- investment tracking
- bill automation
- recurring detection
- tax reporting
- multi-user support
- financial forecasting
- expense export

These were excluded to keep the MVP focused on behavior validation.

---

## Success Criteria

Savvy MVP is successful if users can:

- log expenses in under 3 seconds
- ask spend questions naturally
- get useful category-level visibility
- understand monthly spend without dashboards
- feel more in control of spending with less effort

The MVP is not optimizing for accounting accuracy.  
It is optimizing for low-friction financial awareness.
